Aplicativo Web de Conscientização sobre o Uso do ChatGPT

Desenvolvi este projeto como parte do curso de Sistemas para Internet, com o objetivo de promover a conscientização sobre o uso responsável da inteligência artificial, especialmente do ChatGPT. O sistema foi construído com foco educacional e informativo. Objetivo: Criar um app que oriente usuários sobre as possibilidades, limites e cuidados ao utilizar ferramentas de IA generativa como o ChatGPT. Backend: Desenvolvido com Flask, organizando rotas e lógica de negócio em uma estrutura modular e escalável. Frontend: Interfaces responsivas usando HTML, CSS e JavaScript, com navegação simples e intuitiva. Banco de dados: PostgreSQL integrado via SQLAlchemy para armazenar conteúdos, usuários e interações. Funcionalidades principais: Exibição de conteúdos informativos e notícias sobre IA. Área de administração para gerenciar postagens e usuários. Cadastro/login com autenticação segura. Estrutura pronta para expansão com APIs externas ou frontend em Flutter. Este projeto representa bem minha capacidade de unir habilidades técnicas com propósito social, usando a tecnologia como ferramenta de conscientização.
